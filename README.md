# 📌 EMS_01 – Employee Management System (Module)

## 📖 Overview  
EMS_01 is a lightweight and modular Employee Management System (EMS) designed to manage employee records efficiently. It can be easily integrated into larger applications and serves as a reusable component for handling employee data.

---

## 🚀 Features  
- Add new employee records  
- View employee details  
- Update employee information  
- Delete employee records  
- Search and filter employees  
- Structured data management  

---

## 🛠️ Tech Stack  
- Frontend: HTML, CSS, JavaScript  
- Backend: Python / Django  
- Database: SQLite / MySQL  

---

EMS_01/

## ⚙️ Installation & Setup  

1. Clone the repository  
```
git clone https://github.com/your-username/EMS_01.git
```

2. Navigate to the project folder  
```
cd EMS_01
```

3. Install dependencies  
```
pip install -r requirements.txt
```

4. Run the application  
```
python manage.py runserver
```
